# 🧠 Restaurant Finder AI App

This is a simple AI-powered LangChain app that takes a **locality** as input and returns:

- 🔍 A plausible **restaurant name**
- 📋 A basic **sample menu**
- ⚡ Powered by Together AI + LangChain



## 🚀 Tech Stack

- LangChain 🦜🔗
- Together AI (LLM API)
- Python
